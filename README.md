Руководство пользователя веб-приложения «AParts»
Версия: 1.0
Назначение: Система предназначена для автоматизации процессов продажи автомобильных запчастей, включая каталогизацию, управление корзиной и отслеживание статуса заказов.

1. Общее описание системы
Система AParts представляет собой веб-приложение, работающее в браузере. Интерфейс построен на основе адаптивной верстки (Bootstrap), что позволяет работать с сайтом как с персонального компьютера, так и с мобильного устройства.

2. Установка и запуск (для локального развертывания)
Убедитесь, что на компьютере установлена Java 17.

Скачайте файл aparts.jar из раздела Releases.

Запустите приложение командой: java -jar aparts.jar.

Откройте браузер и перейдите по адресу: http://localhost:8080.

3. Функциональные возможности системы
3.1. Работа с каталогом товаров (Гость / Клиент)
Просмотр категорий: На главной странице отображается иерархия категорий (Двигатель, Ходовая часть, Кузов и т.д.).

Поиск и фильтрация: В верхней части экрана расположена строка поиска. Вы можете ввести часть названия или артикул запчасти (например, фильтр масл). Для сужения выдачи используйте фильтр по бренду и диапазону цен в левой колонке.

Пагинация: Внизу страницы каталога расположены кнопки перехода между страницами (по 20 товаров на странице).

3.2. Регистрация и авторизация
Нажмите кнопку «Вход / Регистрация» в правом верхнем углу.

Регистрация: Заполните поля: Имя, Email, Пароль. Пароль должен содержать не менее 6 символов.

Вход: Используйте Email и пароль для входа в личный кабинет.

Восстановление пароля: Если пароль забыт, нажмите ссылку «Забыли пароль?» и введите Email. Инструкция по сбросу придет на указанную почту.

3.3. Личный кабинет клиента
После входа в систему в шапке сайта отобразится ваше имя.

Профиль: Перейдите в раздел «Мой профиль» для редактирования адреса доставки и контактного телефона.

История заказов: В разделе «Мои заказы» отображается список всех оформленных вами заказов с текущим статусом.

3.4. Оформление заказа (Клиент)
Добавьте необходимые товары в корзину.

Перейдите в Корзину (иконка в правом верхнем углу). Проверьте состав и количество.

Нажмите кнопку «Оформить заказ».

Укажите адрес доставки и выберите способ оплаты.

Подтвердите заказ. После этого заказу будет присвоен статус «Новый».

3.5. Отслеживание статуса заказа
В личном кабинете в разделе «Мои заказы» вы можете видеть текущий этап обработки:

Новый: Заказ поступил в систему, ожидает обработки менеджером.

В обработке (собирается): Сотрудник склада комплектует заказ.

Передан в доставку: Заказ передан курьеру или в транспортную компанию.

Доставлен: Заказ получен покупателем.

Отменен: Заказ отменен по инициативе клиента или из-за отсутствия товара.

3.6. Административная панель (Только для роли ADMIN)
Администратор имеет доступ к скрытому разделу сайта /admin:

Управление товарами: Добавление новых позиций (название, цена, остаток), редактирование карточек товаров, удаление позиций.

Управление персоналом: Добавление учетных записей для работников склада с ролью «Работник».

3.7. Интерфейс работника склада (Роль WORKER)
Смена статусов заказа: Сотрудник видит список активных заказов. Для изменения статуса достаточно нажать соответствующую кнопку («Начать сборку», «Передать в доставку»).
